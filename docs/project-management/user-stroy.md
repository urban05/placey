# User Stories

## Format
> As a **[persona]**, I want to **[action]**, so that **[goal/benefit]**.

---

## 🗺️ Core Map Features

**US-01**
As a **user**, I want to see an interactive map of third places in Wilhelmsburg,
so that I can get a quick overview of what is available near me.

**US-02**
As a **user**, I want to filter places by properties (e.g. free of charge, FLINTA*-friendly, quiet, youth-oriented),
so that I only see places that are relevant to me.

**US-03**
As **Mia**, I want to see which places are marked as FLINTA*-friendly,
so that I can feel safe going there without worrying about the atmosphere.

**US-04**
As **Jonas**, I want to read community comments describing the vibe of a place,
so that I can decide whether it suits my personality before I visit.

**US-05**
As **Jonas**, I want to browse the map without creating an account,
so that I can explore options without committing to anything upfront.

---

## 📍 Place Entries & Discovery

**US-06**
As **Mia**, I want to add a place I discovered to the map,
so that other young people in my neighborhood can find it too.

**US-07**
As a **user**, I want to see up-to-date information about a place (opening hours, tags, description),
so that I don't show up somewhere that is closed or unsuitable.

**US-08**
As a **user**, I want to see events linked to a place,
so that I have a reason to visit at a specific time.

**US-09**
As a **moderator**, I want to review and approve new place submissions,
so that the map stays accurate and trustworthy.

---

## 👤 Account & Community

**US-10**
As **Mia**, I want to register quickly without needing an ID or lots of personal data,
so that I can participate in the community without barriers.

**US-11**
As a **user**, I want to upvote or downvote a place,
so that the most useful and well-loved spots become more visible.

**US-12**
As a **user**, I want to leave a comment on a place,
so that I can share my experience with others.

**US-13**
As **Jonas**, I want to save places to a personal list,
so that I can plan visits without having to find them again on the map.

---

## 🏅 Collection & Engagement (Feature Creep)

**US-14**
As **Mia**, I want to "check in" at a place using GPS or a QR code,
so that I can collect it and feel rewarded for going out.

**US-15**
As a **user**, I want to collect visited places like trading cards,
so that exploring Wilhelmsburg feels fun and motivating.

**US-16**
As **Jonas**, I want to check in manually without GPS if I prefer,
so that I can still participate even if I am uncomfortable sharing my location.

---

## 📱 Mobile & Accessibility

**US-17**
As **Mia**, I want the app to load quickly on my older Android phone with a slow connection,
so that I am not frustrated before I even find a place.

**US-18**
As a **user**, I want the interface to work well on mobile without pinching or zooming,
so that I can use the map comfortably on the go.

**US-19**
As **Jonas**, I want to use the platform in a calm, uncluttered interface,
so that I do not feel overwhelmed when looking for information.

---

## 🔒 Privacy & Trust

**US-20**
As a **user**, I want to know that my data is handled according to GDPR,
so that I can use the platform without worrying about privacy.

**US-21**
As **Jonas**, I want to be able to check in manually instead of sharing live GPS data,
so that I stay in control of my location information.
